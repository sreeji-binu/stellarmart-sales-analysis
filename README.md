# 🛒 StellarMart Sales Analysis

## 📊 Project Overview
This project analyzes retail sales data from StellarMart to identify trends, patterns, and business opportunities. The dashboard provides insights into category performance, regional sales distribution, and top-performing products.

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling products
- Evaluate category-wise contribution
- Understand regional trends
- Track yearly growth patterns

## 📁 Dataset Features
- Sales data
- Product categories
- Regional information
- Year-wise trends

## 📊 Key Analysis
- Category Analysis
- Regional Analysis
- Yearly Trend Analysis
- Top Products Identification

## 🛠 Tools Used
- Power BI / Excel
- Data Cleaning & Transformation
- Data Visualization

## 📌 Key Insights
- Identified high-performing product categories
- Highlighted top revenue-generating regions
- Observed seasonal/yearly sales trends

## 🚀 Outcome
This project helps in making data-driven business decisions and improving sales strategies.
